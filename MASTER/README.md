# RCCG Christ the Anchor Parish - Multi-Page Website

## Overview
This is a complete multi-page website for RCCG Christ the Anchor Parish (FCT Province 17). The site features a professional design, responsive layout, and multiple pages for different sections of the church's ministry.

## File Structure

```
MASTER/
├── index.html           # Home page with hero slideshow and quick links
├── about.html           # About section with mission, vision, values, and three pillars
├── services.html        # Weekly services, activities, and service times
├── events.html          # Upcoming events, calendar, and event carousel
├── ministries.html      # Ministry departments and ways to serve
├── gallery.html         # Photo gallery from worship and community activities
├── giving.html          # Tithes, offerings, and giving methods
├── contact.html         # Contact form and information
├── style.css            # Shared stylesheet for all pages
└── script.js            # Shared JavaScript for all pages
```

## How to Use

1. **Open the site**: Open `index.html` in a web browser
2. **Navigate between pages**: Use the navigation menu at the top
3. **Mobile menu**: Click the hamburger icon on mobile devices
4. **Smooth scrolling**: Click navigation links for smooth scrolling
5. **Submit forms**: Contact forms have validation and error checking

## Customization

### Colors
Edit the CSS variables in `style.css` (lines 1-20):
```css
:root {
    --primary-green: #006400;
    --primary-gold: #FFD700;
    --primary-blue: #1a365d;
    /* etc */
}
```

### Content
Edit the HTML files directly to update:
- Church address and contact info
- Service times
- Ministry names and descriptions
- Bank account details for giving
- Social media links

### Images
All images use placeholder URLs from Unsplash. Replace with actual church photos:
- Search `https://images.unsplash.com/` URLs in HTML files
- Replace with your own image paths

## Maintenance Tips

1. **Update service times** in all pages when schedules change
2. **Keep footer info consistent** across all pages
3. **Update social media links** in all pages (search for `href="#"`)
4. **Add real bank details** for the giving page
5. **Replace placeholder images** with actual church photos
6. **Update contact information** in header and footer
